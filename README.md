# Flutter AI Engineer Roadmap

This repository documents my journey from Flutter Developer to AI Application Engineer.

## Goal

Become an AI Application Engineer by building real-world AI-powered applications using:

- Flutter
- Python
- FastAPI
- Gemini API
- OpenAI API
- Claude API
- LangChain
- LangGraph
- RAG
- Vector Databases
- MCP
- AI Agents
- Computer Vision

## Learning Method

- Learn
- Build
- Deploy
- Document

Every topic includes practical projects instead of theory only.
